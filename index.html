Introduction to C

C is a general-purpose programming language developed by Dennis Ritchie at Bell Labs in 1972. It is one of the most widely used programming languages and forms the basis for many other languages like C++, Java, and Python.

C is known for its efficiency, portability, and flexibility, making it ideal for system-level programming, embedded systems, and applications requiring high performance.

printf("Hello, World!");
Data Types

C has several basic data types, including:

  • int - Integer type
  • float - Floating point type
  • char - Character type
  • double - Double precision floating point type

Each data type in C has a specific size and range of values it can hold, which may vary depending on the system architecture.

int age = 25;
Control Structures

C provides several control structures to manage the flow of execution in programs, including:

  • if - Conditional statement
  • switch - Multi-way branch statement
  • for - Looping statement
  • while - Looping statement

Control structures are essential for making decisions, repeating tasks, and branching logic in C programs.

for(int i = 0; i < 10; i++) { printf("%d ", i); }
Functions in C

Functions are blocks of code that perform specific tasks. In C, functions help in modularizing the code, making it reusable and easier to manage.

Every C program must have a main() function, which is the entry point of the program.

int add(int a, int b) { return a + b; }
Pointers and Arrays

Pointers are variables that store memory addresses, allowing for dynamic memory allocation, efficient array manipulation, and complex data structures.

Arrays are collections of elements of the same type, stored in contiguous memory locations. Pointers can be used to traverse and manipulate arrays efficiently.

  • int arr[5]; - Declares an array of 5 integers
  • int *ptr = arr; - Points to the first element of the array
int *ptr = arr;
for(int i = 0; i < 5; i++) { printf("%d ", *(ptr + i)); }
